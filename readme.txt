== Description ==

Ayyash Studio offers you 20+ premade, super-fast professionally developed and optimized templates for the Elementor Page builder. You just need to select the demo, choose the color and typography according to your requirements and import it to go live with the full template. Gutenberg support will be added soon.

The Ayyash Studio plugin enables you to import a full demo to create a professional website for your needs. The user interface of the plugin makes it easier for you to customize and import demos.

The color preset features will allow you to choose from different predefined color presets. You can also select different typography and logo to customize the demo and import it. The advanced search filters allow you to find the suitable templates you need.

== Frequently Asked Questions ==

= Will “Ayyash Studio” work with any theme? =

Currently, the Ayyash Studio plugin only works with the [Ayyash](https://wordpress.org/themes/ayyash/) theme for now. More theme will be available in the future.

= Are all the templates free? =

Currently, all the published templates are free, and we are working on some premium quality templates.

= Can I import the demo to my existing website? =

Yes, you can import the demo on your existing website. You can choose to import with/without demo content.

= Does it overwrite existing content? =

Yes and no. If you import more than once, you can choose to delete old imported (dummy) contents.
